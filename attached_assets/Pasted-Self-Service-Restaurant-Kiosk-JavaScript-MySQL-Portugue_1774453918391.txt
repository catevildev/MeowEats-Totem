Your task is to design and generate a complete self-service ordering kiosk system similar to McDonald's, Burger King, and fast-food chains.

The entire system must be:

Written in JavaScript only (NO TypeScript)
Database must be MySQL only
Entire UI must be Portuguese (Brazil)
Designed for Touchscreen self-service kiosks
Focus on speed and usability
